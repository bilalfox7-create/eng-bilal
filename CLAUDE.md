## نظرة عامة
نظام إدارة داخلي لشركة الرائد (ليبيا). بيتتبع المهندسين في 4 محافظات (درنة/البيضاء/بنغازى/طبرق):
**الإعاشة** الشهرية + **الحضور والانصراف** + **العهد والمصروفات** + **إجازات مصر** + **الهيكل التنظيمي** + التقارير.
الواجهة عربية (RTL)، والتعامل مع المستخدم (بلال) باللهجة المصرية.

## 💾 الداتا والنسخ الاحتياطي والاستعادة  ← الأهم
بعد كارثة Railway (مايو 2026 — الاستضافة ماتت والداتا اتحبست)، الداتا دلوقتي في **3 أماكن**:
1. **Turso** (libSQL cloud) — القاعدة الحيّة، منفصلة عن الاستضافة.
2. **GitHub — ريبو خاص `bilalfox7-create/eng-bilal-backups`** — ملف `backup-latest.json` فيه كل جدول `months`. بيتحدّث:
   - **تلقائيًا كل يوم** الساعة 2 صباحًا (`server.js` → `runBackup`).
   - **يدويًا** عبر `POST /api/github-backup` (admin) — **التطبيق** هو اللي بيرفع.
   - `git history` بيحتفظ بكل النسخ القديمة (ترجع لأى تاريخ).
3. **تنزيل يدوي** من صفحة النسخ الاحتياطي جوه الموقع.
- **الاستعادة:** `PUT /api/data` (استبدال كامل) أو `PUT /api/months/:key` مع `force:true` (شهر واحد). شكل الداتا في `backup-latest.json` هو `{ backedUpAt, monthsCount, months:{ "YYYY-MM": {...} } }`.
- ⚠️ **مصنّف الأمان بيمنع Claude إنه يـ push داتا فيها بيانات شخصية لريبو خارجي.** عشان كده **خلّي التطبيق هو اللي يعمل الباك أب** (نادِ الراوت أو استنى اليومي) — متحاولش ترفع الداتا من جهازك مباشرة؛ هتتمنع.

## الـ Stack وخريطة الملفات
- **الباك إند:** Express 4 + `@libsql/client` (async).
  - `db.js` — اتصال libSQL + دوال async: `get/all/run/exec/batchWrite` + `initDb()` (إنشاء الجداول + بذر المستخدمين).
  - `routes/auth.js` — login/logout/me/change-password (bcryptjs + sessions + rate limiter).
  - `routes/api.js` — months CRUD، leaves، org-chart، logo، fixed-rents، users، activity، backups، `github-backup`. كله async/await. صلاحيات: admin (كامل) / **hr (= admin كامل تمامًا — قرار بلال؛ الكلاينت بيطبّع `role hr→admin` عند login/me، والسيرفر `adminOnly` بيسمح لـ hr + فحص القراءة-فقط في `PUT /months` مابيمنعش hr)** / province (تعديل محافظته بس — merge) / viewer (قراءة فقط).
  - `server.js` — express + static + sessions + routes + `app.get('*')→index.html` + بدء async (`await initDb()` ثم listen) + نسخة احتياطية تلقائية كل 24 ساعة (محلي + GitHub).
  - `github-backup.js` — يبني JSON من جدول `months` ويرفعه لـ `backup-latest.json` عبر GitHub Contents API (PUT بـ base64 + sha؛ throttle 5 دقايق إلا `force`).
  - `scripts/check-index.cjs` — فحص parse لـ `public/index.html` (بيستخدمه الـ pre-commit hook).
- **الفرونت إند:** **ملف واحد** `public/index.html` (~530 ألف حرف). React 18 UMD (بدون JSX) — `h()` = `React.createElement`. كله في `<script>` واحد كبير. خطوط: Cairo + Cormorant Garamond + Bebas Neue.

## نموذج البيانات (جدول `months`)
- `months` مفتاحه `"YYYY-MM"` (+ مفتاح خاص `"leaves"`). كل صف: `{data, cfg, savedAt, expenses, attendance}`.
- `data` = `{ "اسم المحافظة": [ مهندس, ... ] }`. الأسماء: `محافظة درنة` / `محافظة البيضاء` / `محافظة بنغازى` / `محافظة طبرق`.
- **المهندس:** `{id, name, spec, d1, d2, extra?, egyptLeaves?, departedAt?}`. الـ ids: درنة d1..d10 · بيضاء b1..b12(+) · بنغازى g1..g7 · طبرق t1..t3(+).
- **الإعاشة:** `d1` = أيام النصف الأول (0-15)، `d2` = أيام النصف الثاني (0-15)، `extra` = أيام قديمة يدوية. المبلغ = `الأيام × (cfg.rate/30)` بالدولار؛ والجنيه = المبلغ × `cfg.egp`. (دالة `calc()` ~سطر 1785). الافتراضي: `rate=300` (= $10/يوم، $300/شهر)، `egp=52.5`.
- **expenses** = `{custody:[], items:[], transfers:[]}`:
  - عهدة: `{id, desc, source, responsibleId, prov, usd, lyd, egp, date, note}`.
  - مصروف/إيجار: `{id, desc, type:'مصروف'|'إيجار', category, prov, usd, lyd, egp, date, note, custodyId, fixedRentId?}`. الإيجار "مدفوع" فقط لو `type==='إيجار' && fixedRentId===r.id && date===dueDate`.
  - تحويل عملة: `{id, fromCurrency, toCurrency, usd, lyd, egp, rate, date}` — المبلغ بيتقرا من الحقل اللي اسمه نفس العملة.
- **fixed_rents** (app_config): `[{id, desc, day, lyd, usd, prov}]` — قوالب إيجار ثابتة، مستحقة شهريًا في يوم `day`.
- **attendance** = `{ "engId": { "day": {s, in?, out?} } }`. الحالة `s`: `P`=حاضر · `A`=غياب · `AL`=إجازة · `SL`=مرضية · `EG`=فى مصر (خلية يدوية) · `EGL`=إجازة مصر (**محسوبة تلقائيًا من `egyptLeaves` — مش خلية مخزّنة**) · `H`=عطلة · `TXR`=متأخر. الجمعة = عطلة أسبوعية.
- **egyptLeaves** (على المهندس): `[{id, departureDate, returnDate, notes, createdBy, createdAt}]`. في إجازة لو `departureDate <= التاريخ < returnDate` (returnDate = يوم الرجوع، حصري؛ null = مفتوحة). هي اللي بتلوّن خلايا EGL برتقالي.

## ⚠️ قواعد حرجة (التزم بيها)
1. **افحص جافاسكريبت قبل أى رفع/commit لـ index.html.** أى تعريف مكرر (`const`/`let`/`function`) في السكربت الكبير الواحد **بيبيّض الموقع المنشور بالكامل**. الفحص: استخرج محتوى الـ `<script>` و`new Function(body)` — لازم يـ parse نضيف. (فيه pre-commit hook بيعمل ده تلقائيًا.)
2. **اختبر الموقع Live بعد كل نشر، بـ 3 أدوار** (admin + province + viewer) وبتقرير ✅/⚠️/❌. راعِ الـ cold start (~50 ثانية).
3. **العهدة والمصروفات منفصلتين تمامًا.** العهدة = سجل استلام لكل عملة (رصيد مجمّع). المصروفات بتخصم من الرصيد المجمّع. **ممنوع نهائيًا** أى UI/عمود/قسم "من عهدة" لكل مصروف. "دفعت من جيبك" = رصيد مجمّع سالب (بانر أحمر).
4. **مفيش كشف تلقائي للمستحقات المرحّلة** في كشف الإعاشة — متفحصش الشهور السابقة لـ d1/d2 غير المدفوعة؛ المستخدم بيقرر يدويًا.
5. **3 عملات في كل حتة:** `$` (دولار) · `د.ل` (دينار ليبي) · `ج.م` (جنيه مصري). خليهم التلاتة في الفورمات والجداول والـ PDF والأرصدة.
6. **الدقة أهم حاجة** ("مش عايز بيانات غلط"). قبل أى كتابة على الداتا الحيّة، اعرض القيم (أسماء/أرقام/تواريخ) بوضوح للمراجعة. **المجاميع ممكن تعدّي وفيه خطأ في صف** (بلال مسك أخطاء كتير كده) — راجع معاه، متعتمدش على الـ checksums لوحدها. لما تقرا أرقام من صور/PDF، كبّر/قُص لتقرا الأرقام بالظبط (٠-٩).
7. **بلاش نوافذ أسئلة منبثقة (AskUserQuestion).** اسأل في الشات بنص عادي، وادّي اقتراح افتراضي يقدر يوافق عليه بسرعة.
8. **حفظ مستخدم المحافظة لازم يدمج *كل* تعديلات محافظته** — مش بس `d1/d2/paid`+الحضور. كمان: المهندسين كاملين (إضافة جديد + تعديل اسم/تخصص + `egyptLeaves` + `egyptTransfer`) + العهد والمصروفات بتاعة محافظته (`prov===userProv`). لو ضيّقت دمج فرع `province` في `PUT /api/months/:key`، شغل المحافظات **بيضيع بصمت** ومايوصلش للأدمن (كانت مشكلة حقيقية، اتصلحت 2026-06-01). شرطا الأمان في الدمج: **ماتلمسش محافظة تانية**، و**ماتمسحش مهندس موجود** (دمج/إضافة بس).
9. **تقارير PDF — متحطّش `radial-gradient` ولا تدرّجات بأطراف `transparent` في خلفيات الطباعة.** Chrome بيحوّلها لـ PDF ShadingType 1 (function-based)، وقارئات PDF كتير (زى pdf.js) بترسمها **وردي/ماجنتا** (لون خطأ) فوق أى خلفية تحتها. استخدم **لون صلب** + `linear-gradient` بحدود لونية صلبة (axial = ShadingType 2 = مدعوم). الغلاف كان بيطلع وردي بسبب ده — اتصلح 2026-06-01 (شيلنا الـ radials من `.cover`). للاختبار: `chrome --headless --print-to-pdf` ثم رندر بـ pdf-to-img وشوف بعينك.

## التعامل مع بلال
- مصري، ودّي وبسيط. ردود مركّزة؛ جداول/نقط قصيرة للداتا اللي محتاج يراجعها.
- **بيفوّض ويثق** ("اتصرف"، "سيبها عليّا") — خد المبادرة في الشغل التقني/الممل بدل ما تخليه هو يعمله. بموافقته بيسيبني أسوق المتصفح بنفسي (لوحة Render).
- **لما هو لازم يعمل خطوة في لوحة تحكم:** خطوة واحدة واضحة في المرة + "صوّرلي الشاشة". متكتبش قوايم طويلة (بيتلخبط).
- **الداتا مقدّسة.** متخاطرش بيها أبدًا؛ طمّنه صراحةً لما حاجة مش هتأثر عليها.
- **متبالغش في الوعود.** لو حاجة مش هتتعمل ببلاش أو مش هتترجع، قول بصراحة وادّي الطريق الواقعي.
- بيهتم بالشكل (mockups، تعديلات). تدرّجات **بنفسجي→وردي** للوحات/الإكسل، و**كحلي (#0A0E1A) + ذهبي (#D4A538)** للهيكل/التقارير الرسمية. خطوط Cormorant Garamond + Bebas Neue للتقارير الفخمة.
